# Cognitive Concepts - Interaction Design

## Interactive Elements

### 1. Concept Cards Hover Enhancement
- **Trigger**: User hovers over any concept card
- **Effect**: Subtle glow effect with primary color (#58A6FF) border, slight upward translation (4px)
- **Duration**: 200ms smooth transition
- **Purpose**: Provide visual feedback and create depth without distraction

### 2. Progressive Disclosure
- **Trigger**: User clicks on concept card title
- **Effect**: Card expands vertically to reveal additional details or examples
- **Behavior**: Only one card can be expanded at a time (accordion-style)
- **Animation**: Smooth height transition over 300ms
- **Purpose**: Allow users to explore concepts in-depth while maintaining clean initial view

### 3. Smooth Scroll Navigation
- **Trigger**: User scrolls the page
- **Effect**: Subtle parallax on background geometric patterns
- **Speed**: Background moves at 0.5x scroll speed
- **Purpose**: Add visual depth and create engaging scrolling experience

## Non-Interactive Elements

- No forms, questionnaires, or input fields
- No guides, manuals, or self-help materials
- No quizzes or interactive assessments
- Focus is purely on elegant presentation of existing content
